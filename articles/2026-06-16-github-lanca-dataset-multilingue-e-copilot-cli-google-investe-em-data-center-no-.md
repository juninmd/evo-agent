# GitHub lança dataset multilíngue e Copilot CLI; Google investe em data center no Alabama

**Período analisado:** 15/06/2026 a 16/06/2026

Um novo dataset aberto da GitHub facilita a descoberta de conteúdo técnico em vários idiomas, enquanto o Copilot CLI traz comandos de barra para automatizar fluxos de terminal. Paralelamente, a Google destinou US$ 1,5 bilhão para ampliar sua infraestrutura no Alabama, sinalizando foco geográfico em capacidade de nuvem.

O lançamento do dataset multilíngue pela GitHub fornece uma base rica para treinamento de modelos que atendam a desenvolvedores globais, ao passo que o Copilot CLI traz automação prática para ambientes de terminal.
